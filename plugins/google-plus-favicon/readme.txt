== Description ==

This plugin allows you to generate a [Google+](https://plus.google.com/) favicon for your blog and admin logo included Apple touch icon. Activate the plugin, and it will add G+ photo to your blog template and admin panel automatically.

== Installation ==
Support Wordpress and Wordpress Multisite

= Standard Multisite Plugins installation. =

1. Upload `google-plus-favicon.php` to the `/wp-content/mu-plugins/` directory

= Standard Plugins installation. =

1. Upload `google-plus-favicon.php` to the `/wp-content/plugins/` directory.

2. Activate the plugin through the `Plugins` menu in WordPress

3. Enter G+ account ID in `General Settings`

You can disabled the web site icon option in General Settings. Then the favicon will turn off.
