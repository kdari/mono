How to install:
1. Upload this folder to your themes folder at: wp-content/themes
2. Log into WordPress administration and click on "Presentation"
3. Look for the Gluttony image and click on it to install / activate this theme
4. Go back to your blog's front page and refresh / reload it

Usage tip:
Within blog posts, assigning the "alignleft" or "alignright" selector to any image will make that image float left or right. For example:

<img src="youimage.gif" class="alignleft" alt="" />
